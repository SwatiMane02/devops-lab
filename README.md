# MyCollege ERP 

This is the **MyCollege ERP System**, built with **React + TypeScript (TSX)**.

## Features
- Student Dashboard, Attendance, Fees, and Results pages
- Faculty Assignments, Attendance, and Grades pages
- Login and Registration pages with validation
- Role-based access for Higher Authority, Nodal Officers, and Processing Staff

## Tech Stack
- React + TypeScript (TSX)
- React Hook Form + Zod for form validation
- Vite for development server

## Getting Started
1. Clone the repository
   ```bash
   git clone https://github.com/<your-username>/mycollege-erp.git
   cd mycollege-erp/frontend
   ```

2. Install dependencies
   ```bash
   npm install
   ```

3. Start the development server
   ```bash
   npm run dev
   ```

4. Open in browser
   ```
   http://localhost:5173
   ```
